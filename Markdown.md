# Introduction to Markdown

Markdown is a **lightweight markup language** used to format text in a simple and readable way. It is widely used on platforms like GitHub to create documentation files such as *README.md*.

## Why Use Markdown?

Markdown makes it easy to write content that is both **clean** and *structured* without needing complex formatting tools. Developers prefer it because it keeps files readable even in plain text form.

### Key Features

- You can create **bold text** using double asterisks.
- You can write *italic text* using single asterisks.
- Headings help organize content clearly.
- It works seamlessly on **GitHub repositories**.

In summary, Markdown is a *powerful yet simple* tool for writing formatted content efficiently.
